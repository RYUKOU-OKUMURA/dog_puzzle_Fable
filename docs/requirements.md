# 要件定義書 — しばちゃんのおさんぽパズル(仮)

## 1. プロジェクト概要

| 項目 | 内容 |
|---|---|
| プロジェクト名 | しばちゃんのおさんぽパズル(仮) |
| ジャンル | 3Dジオラマ箱庭風・経路パズルゲーム |
| プラットフォーム | Webブラウザ(PC / タブレット対応) |
| 対象ユーザー | 小学校3〜4年生(8〜10歳)、特に犬好きの女の子 |

## 2. コンセプト

**「柴犬と一緒にお散歩して、世界中の犬と友だちになる」**

- 主人公は柴犬の「しばちゃん」(ナビゲーター兼相棒)
- 道パネルをはめてお散歩ルートを完成させる経路パズル
- お散歩のゴールで新しい犬種と出会い、図鑑(思い出アルバム)に記録される
- パズル(論理的思考)+ 犬の知識 + 世界地理の学習要素を自然に組み込む

## 3. ターゲットユーザーへの配慮

- 漢字にはふりがなを付ける、またはひらがな・カタカナ中心の表記
- カメラ操作は不要(固定のアイソメトリックビュー)にして操作を簡単に
- タップ / クリック / ドラッグだけで完結する操作体系
- 失敗してもペナルティなし。何度でもやり直せる
- かわいさ重視のビジュアルトーン(パステル調、丸みのあるデザイン)

## 4. ゲームフロー

1. **ステージ選択**: 世界地図から行きたい場所(ステージ)を選ぶ
2. **パズルフェーズ**: ジオラマ風の街に欠けた道があり、道パネルをはめてスタート(おうち)からゴールまでのルートを完成させる
3. **お散歩フェーズ**: ルートが完成すると、しばちゃんが完成した道をテクテク歩く(ごほうび演出)
4. **出会いフェーズ**: ゴールで新しい犬種と出会い、記念写真を撮る
5. **図鑑登録**: 犬種の豆知識と完成したジオラマのミニチュアが「おさんぽずかん」に保存される

## 5. 機能要件

### 5.1 パズル(コア機能)

- 道パネルの種類: まっすぐ / 曲がり角 / T字路 / 橋 / 階段(段差移動)
- パネルはドラッグ&ドロップ(またはタップで選択→タップで配置)ではめる
- 配置済みパネルの回転・取り外しが可能
- ルートがつながっているかの自動判定
- 「おさんぽスタート」ボタンで判定 → 成功ならお散歩フェーズへ、失敗なら柴犬が途中で止まって首をかしげる(やさしいフィードバック)

### 5.2 難度設計(ステージが進むと追加される条件)

| レベル | 追加要素 |
|---|---|
| 初級 | 道をつなぐだけ |
| 中級 | おやつを全部拾って通る / 水たまりを避ける |
| 上級 | 使えるパネルの枚数制限 / 橋で道を交差させる / 階段で高低差を使う |

### 5.3 図鑑(おさんぽずかん)

- 出会った犬種ごとに1ページ
- 掲載内容: 犬種名(ふりがな付き)、写真(3Dモデルのスクリーンショット)、原産国、性格、大きさ、豆知識
- ステージの舞台は犬種の原産国と対応(例: コーギー=イギリス風の街並み、チワワ=メキシコ風)
- 収集率の表示(「12/30 しゅるい」など)でコレクション欲を刺激
- 柴犬は最初から図鑑に登録済みの特別な相棒

### 5.4 その他

- 進捗のローカル保存(ブラウザのlocalStorage。アカウント登録不要)
- BGM / 効果音のON/OFF
- 犬の歩行アニメーション(かわいさ最優先)

## 6. 非機能要件

- タブレット(iPad等)のタッチ操作で快適に遊べること
- 通信不要でプレイ可能(初回ロード後はオフラインでも動作が理想)
- 1ステージ5〜10分で完結(子供の集中時間に合わせる)
- 個人情報の収集なし、外部リンクなし、課金なし

## 7. スコープ

### フェーズ1(プロトタイプ)

- 1ステージのみ(日本の街+柴犬)
- 道パネル3種(まっすぐ / 曲がり角 / T字路)
- パズル → お散歩 → 図鑑保存の一連の流れ

### フェーズ2以降(将来)

- ステージ追加(世界の国々+犬種)
- 橋・階段など立体パネル
- おやつ収集などの追加ルール
- きせかえ(しばちゃんのバンダナ等)などのごほうび要素
