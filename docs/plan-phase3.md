## M13: こうかおん(効果音3点 + おと ON/OFF)

BGMと追加効果音はスコープ外(未承認)。今回は3点のみ先行し、子供の反応を見る。

### 仕様

#### 効果音3点 — Web Audio API 合成のみ

音色・パラメータは [prototypes/audio-demo.html](./prototypes/audio-demo.html) から移植する(**勝手に変えない**)。

| 名前           | タイミング                                 | プロトタイプ  |
| -------------- | ------------------------------------------ | ------------- |
| ポン           | パネルをスロットに置いたとき               | `SFX.pon`     |
| ぱくっ         | お散歩中、しばちゃんがおやつを通過した瞬間 | `SFX.paku`    |
| ファンファーレ | クリア成功の演出開始時                     | `SFX.fanfare` |

共通:

- 初回ユーザータップで AudioContext を生成・resume(iPad 自動再生制限対応)
- DynamicsCompressor 経由で音割れ防止。プロトタイプの master 構成を踏襲
- 音声ファイル・howler.js 等の**依存追加は禁止**
- 回転音・失敗音などの追加もしない(3点で子供の反応を見るため)

#### おと ON/OFF 設定

- トグルは1個、名前は「おと」。初期値 ON
- プロフィール別に保存(`save/` の既存形式への**追記フィールド**。旧セーブは欠損時 ON で読める後方互換。破壊的変更禁止)
- UI: タイトル画面に小さなトグルボタン(例: 🔊「おと ON」/ 🔇「おと OFF」)。文言はひらがな。design-guide の既存パレット・ボタン様式のみ使用、新しい色・装飾を足さない
- この構成(タイトル画面・トグル1個・ひらがな文言)までは承認済み。逸脱したくなったら実装せず人間に確認

#### アーキテクチャ

- 新モジュール `src/audio/`(例: `sfx.ts`)。他層に依存しない・DOMも触らない。「鳴らす関数」と有効フラグだけを公開
- 呼び出しは `game/state.ts` の状態遷移と `main.ts` の配線に集約。`scene/`・`core/` から直接呼ばない
- ON/OFF分岐は audio モジュール内で持ち、呼び出し側に散らさない
