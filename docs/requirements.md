# 要件定義書 — しばちゃんのおさんぽパズル(仮)

## 1. プロジェクト概要

| 項目 | 内容 |
|---|---|
| プロジェクト名 | しばちゃんのおさんぽパズル(仮) |
| ジャンル | 3Dジオラマ箱庭風・経路パズルゲーム |
| プラットフォーム | Webブラウザ(PC / iPad等タブレット)。GitHub Pages で配信し、PWA としてホーム画面から起動 |
| 対象ユーザー | 自分の子供たち(小学校3〜4年生中心)。**完全に家庭内利用。一般公開・ローンチはしない** |

### 個人利用ゆえの前提

- リポジトリは GitHub Pages(無料プラン)の都合で public になる。**子供の実名など個人情報はコード・コミット・ステージデータに一切含めない**(プレイヤー名は localStorage のみ)
- 収益化・アカウント・外部通信・アクセス解析は一切なし

## 2. コンセプト

**「柴犬と一緒にお散歩して、世界中の犬と友だちになる」**

- 主人公は柴犬の「しばちゃん」(ナビゲーター兼相棒)
- 道パネルをはめてお散歩ルートを完成させる経路パズル
- お散歩のゴールで新しい犬種と出会い、図鑑(思い出アルバム)に記録される
- パズル(論理的思考)+ 犬の知識 + 世界地理の学習要素を自然に組み込む

## 3. ターゲットユーザーへの配慮

- 漢字にはふりがなを付ける、またはひらがな・カタカナ中心の表記
- カメラ操作は不要(固定のアイソメトリックビュー)にして操作を簡単に
- タップ / クリック / ドラッグだけで完結する操作体系
- 失敗してもペナルティなし。何度でもやり直せる
- 行き詰まったら「しばちゃんヒント」で救済(→ 5.3)
- かわいさ重視のビジュアルトーン(パステル調、丸みのあるデザイン)。**現行UIのクオリティを最低ラインとする**

## 4. ゲームフロー

1. **プロフィール選択**: 起動時に「だれが あそぶ?」で自分のプロフィールを選ぶ
2. **ワールド / ステージ選択**: 世界地図からワールド(国)→ ステージを選ぶ
3. **パズルフェーズ**: ジオラマ風の街の欠けた道に道パネルをはめて、スタート(おうち)からゴールまでのルートを完成させる
4. **お散歩フェーズ**: ルートが完成すると、しばちゃんが完成した道をテクテク歩く(ごほうび演出)
5. **出会いフェーズ**: ゴールで新しい犬種と出会い、記念写真を撮る
6. **図鑑登録**: 犬種の豆知識と完成したジオラマのミニチュアが「おさんぽずかん」に保存される

## 5. 機能要件

### 5.1 パズル(コア機能)

- 道パネルの種類: まっすぐ / 曲がり角 / T字路 / **橋(道の交差。ワールド4で解放)**
- パネルはドラッグ&ドロップ(またはタップで選択→タップで配置)ではめる
- 配置済みパネルの回転・取り外しが可能
- ルートがつながっているかの自動判定。**空きスロットは全部埋めなくてもよい**(スタート→ゴールがつながればクリア。ダミースロットは埋めても不利にならないが、正解には使わない)
- 「おさんぽスタート」ボタンで判定 → 成功ならお散歩フェーズへ、失敗なら柴犬が途中で止まって首をかしげる(やさしいフィードバック)
- **おやつ収集ステージ**: 盤面に置かれたおやつを全部通るルートでないとクリアにならない(ワールド2で解放)。おやつを全部通れるルートが盤面上に作れていればクリア — **しばちゃんは遠回りしてでもおやつを通る道をえらんで歩く**(近道があっても取り逃さない)。おやつを全部は通れない盤面のときはお散歩に出ず、やさしく知らせる

### 5.2 ワールド構成と難度カーブ(5ワールド × 4ステージ = 20ステージ)

難易度は「ステージが進むと難しくなる」方式。ワールドごとに新要素を1つずつ解放する。

| W | 国・テーマ | 新要素 | 出会える犬(4匹 / 原産国は実際に合わせる) |
|---|---|---|---|
| 1 | にほん(鳥居・桜の街) | なし。道をつなぐだけ。W1-1はチュートリアル | 秋田犬 / 狆(ちん) / 日本スピッツ / 甲斐犬 |
| 2 | イギリス(レンガの家・赤い電話ボックス) | おやつ収集 | コーギー / ビーグル / ブルドッグ / ゴールデンレトリバー |
| 3 | フランス・スイス(石畳・アルプス) | 新ギミックなし。空きマス増・ダミーパネルで複雑化 | プードル / パピヨン / セントバーナード / バーニーズマウンテンドッグ |
| 4 | メキシコ・アメリカ大陸(カラフルな家・サボテン) | 橋(道の交差) | チワワ / ボストンテリア / アラスカンマラミュート / ラブラドールレトリバー |
| 5 | せかいのたび(雪の街・夜の街など変化球) | 全要素の複合・最大盤面 | シベリアンハスキー / サモエド / シーズー / ダックスフンド |

- 図鑑は相棒のしばちゃんを含めて全21種類
- ワールドは前のワールドの全ステージクリアで解放
- ワールド内は **s1 クリアで s2〜s4 が解放**(新要素の導入ステージを必ず最初に通すため)。s2〜s4 は自由順

#### 難度の割り当て(ほね 1〜5)

難度は5段階。ステージ選択画面に 🦴 の数で表示する(★ではなく骨マーク。世界観に合わせる)。
配分は「やさしいステージは最初だけ、歯ごたえゾーンを厚く」— 🦴1×1 / 🦴2×3 / 🦴3×6 / 🦴4×7 / 🦴5×3。

難度を作るレバーは6つ: 空きスロット数 / 盤面サイズ / パネル種の制限(palette) / おやつ数 / ダミースロット(正解に使わない空きマス) / 橋。
パネルの枚数制限はスコープ外(7章)。空きスロットは全部埋めなくてもよいルール(5.1)のため、難度差は「スロット数」と「分岐・ダミーの紛らわしさ」で作る。

| ステージ | 🦴 | 難度の中身 |
|---|---|---|
| W1-1 | 1 | チュートリアル。空き5、直線+曲がりのみの一本道 |
| W1-2 | 2 | 空き5〜6、曲がり多め |
| W1-3 | 2 | 空き7、T字の分かれ道(行き止まり側の釣り分岐) |
| W1-4 | 3 | 空き8〜9、遠回りルートが正解 |
| W2-1 | 2 | **おやつ導入**。おやつ1・空き6 |
| W2-2 | 3 | おやつ2、空き8 |
| W2-3 | 3 | おやつ2、最短路では取れない配置 |
| W2-4 | 4 | おやつ3、空き10 |
| W3-1 | 3 | **ダミースロット導入**(ダミー2) |
| W3-2 | 3 | パネル種の制限(T字なし)+ダミー3で紛らわせる |
| W3-3 | 4 | おやつ2+ダミー3の複合 |
| W3-4 | 4 | 9×9、最長ルート+ダミー4 |
| W4-1 | 3 | **橋導入**。橋1、交差1回だけ |
| W4-2 | 4 | 橋1+おやつ2 |
| W4-3 | 4 | 橋2+ダミー |
| W4-4 | 5 | 橋2+おやつ3 |
| W5-1 | 4 | 10×10、全要素ひかえめ複合 |
| W5-2 | 4 | おやつ+ダミー多め |
| W5-3 | 5 | 全要素の複合(全部乗せ) |
| W5-4 | 5 | 最大盤面・最難関。クリアで完走のお祝い |

- **ノコギリ型カーブ**: 新要素の導入ステージ(W2-1・W4-1)は1段下げて「ルールを覚える回」にする。それ以外はワールド内で右肩上がり
- **時間の上限**: 🦴5でも1ステージ5〜10分(6章)に収める。テストプレイで10分を超えるなら空きスロットを減らして調整する。詰まったときの救済はしばちゃんヒント(5.3)

### 5.3 しばちゃんヒント

- 一定時間解けない(または「おさんぽスタート」失敗が続く)と、しばちゃんが正解ルート上の空きマスの1つに歩いて行ってちょこんと座り、そのマスがきらっと光る
- ペナルティなし。世界観に溶けた救済としてUIボタンは置かない

### 5.4 プロフィール(子供ごとのセーブ分離)

- 起動時に「だれが あそぶ?」画面。プロフィールは最大4つ
- 作成時に なまえ(ひらがな)+ すきな犬アイコン を選ぶ
- 図鑑の収集状況・ステージクリア状況・きせかえの入手/装備・設定をプロフィールごとに保存
- 端末をまたいだ同期はしない(iPadはiPadの記録。サーバーレス方針を維持)

### 5.5 きせかえ

- ワールドをクリアするごとに、しばちゃんのアクセサリー(バンダナ等)を1つ入手(全5種)
- プロフィール画面やステージ選択画面から付け替えでき、お散歩フェーズのしばちゃんに反映される

### 5.6 図鑑(おさんぽずかん)

- 出会った犬種ごとに1ページ
- 掲載内容: 犬種名(ふりがな付き)、写真(3Dモデルのスクリーンショット)、原産国、性格、大きさ、豆知識
- ステージの舞台は犬種の原産国と対応させ、地理の学びにつなげる
- 収集率の表示(「12/21 しゅるい」など)でコレクション欲を刺激
- 柴犬は最初から図鑑に登録済みの特別な相棒

### 5.7 その他

- 進捗のローカル保存(localStorage、プロフィール別。アカウント登録不要)
- 犬の歩行アニメーション(かわいさ最優先)

## 6. 非機能要件

- タブレット(iPad等)のタッチ操作で快適に遊べること
- PWA対応: ホーム画面追加でフルスクリーン起動、初回ロード後はオフラインでも遊べる
- 1ステージ5〜10分で完結(子供の集中時間に合わせる)
- 個人情報の収集なし、外部リンクなし、課金なし

## 7. スコープ

### フェーズ1(プロトタイプ)— 完了

- 1ステージ(日本の街+柴犬)、道パネル3種、パズル → お散歩 → 図鑑保存の一連の流れ

### フェーズ2(今回)

1. ステージ量産の土台(テキスト地図形式のステージ定義)+ GitHub Pages デプロイ + PWA
2. プロフィール機能(セーブデータ移行含む)
3. ワールド選択画面と20ステージ(W1→W5 の順に制作)
4. おやつ収集(W2〜)、橋パネル(W4〜)
5. しばちゃんヒント
6. きせかえ(ワールドクリア報酬)

### フェーズ3以降(将来・見送り)

- BGM / 効果音(ON/OFF設定含む)
- 記念写真の図鑑保存・端末への画像保存
- 階段(高低差)パネル、水たまり回避・パネル枚数制限ルール
- ステージ・犬種のさらなる追加、パズル以外のミニゲーム
